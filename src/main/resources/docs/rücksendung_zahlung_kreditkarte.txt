Sicherer Zahlungsverkehr:
Deine Daten werden verschlüsselt und damit sicher übertragen.

bonprix verwendet die modernsten Verschlüsselungstechniken, um zu verhindern, dass Dritte unberechtigt auf Deine Daten zugreifen, z. B. das Secure Socket Layer (SSL).

SSL ist ein im Internet anerkanntes Standardprotokoll, das einen sicheren Datenaustausch gewährleistet. Dabei werden sämtliche Bestelldaten ausschließlich verschlüsselt an bonprix übermittelt. So auch die Daten Deiner Kreditkarte.

Zusätzlich bietet bonprix den Kunden von Mastercard und VISA das 3D-Secure- Authentifizierungsverfahren an. Damit wird es Unbefugten unmöglich gemacht, Deine Kreditkarte bei den teilnehmenden Shops zu nutzen.

Der Kreditkartenkauf ist aus Gründen der Sicherheit nur online möglich.
